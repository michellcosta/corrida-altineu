# Arquivos de exemplo

## resultados-teste.csv

Arquivo de exemplo para importar resultados em `/admin/chip/results`.

**Antes de importar:**
1. Aplique a numeração em `/admin/chip/numbering` (botão "Aplicar numeração")
2. Os bib numbers 1–20 no CSV devem corresponder a inscrições existentes

Se tiver menos de 20 inscritos numerados, edite o CSV e remova as linhas cujos bib não existem no seu evento.
